# pcf8575-test
Small tests with Rpi like hardware, i2c bus, pcf 8575 and leds...

Just to make leds blinking for hardware and sofware tryouts....


# hardware
* orangepi one plus / raspberrypi
* pcf8575 (i2c port extender 16 ports)
* resistors : 1k, 100k, ...
* 2n2907
* leds
